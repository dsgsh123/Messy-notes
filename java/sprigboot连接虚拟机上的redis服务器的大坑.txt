1. 虚拟机默认是没有开放6379端口的，需要开放端口，关闭防火墙（具体百度），同时CENTOS7.5中要记得使用 nc -lp 6379 &开启监听，这点很关键，因为不监听的话虽然已经开放了端口，但还是连不上

2. 可以使用telnet host port的命令来查看是否可以访问端口

3. redis默认开始保护模式和绑定（bind 127.0.0.1），这样的话远程无法连接，需要把保护模式改为no，并且将绑定注释掉，或者改为0.0.0.0，同时启动redis要使用 redis-server redis.conf来启动服务，redis默认是不会使用配置文件启动服务的，所以如果不指定配置文件，你会发现在配置文件里的修改都没有用
